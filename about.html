About the Portal

This management portal is a simulated dashboard representing how modern colleges run administrative, staff, and student workflows. The system features:

  • Admin Portal: High-level dashboard, student/staff CRUD operations, departments settings, database seeding.
  • Staff Portal: Filtered students listing, student grades and marks entry, daily attendance rolls tracker.
  • Student Portal: Detailed academic scorecard sheet, daily present status history, visual schedule tracker.
  • Client-Side Storage: No databases required; uses browser Local Storage to seed and persist records.
